[About EverAS]

EverAS (AS stands for Application Suite/System) is a web-integrated, cloud based application system, designed to simulate how a command-line interface (CLI) functions and works.

EverAS is designed to run on both desktop and mobile devices, with portability, and simplicity as the priority. This system is made for everyone curious about how a CLI operates — from students and beginners learning computing basics to developers wanting a lightweight, portable environment accessible on desktop and mobile. Unlike traditional operating systems or desktop-only shells, EverAS runs entirely in your browser, combining cloud accessibility, device portability, and modular CLI-style tools in a single suite.

EverAS uses simple commands such as help for displaying important commands, clear for clearing the console screen, and exit for exiting a current feature you opened and nav + feature for navigating to any features.

Currently, EverAS contains different applications known as “features”. These features are the following:

1. Home menu – the default screen for the system. Features a date and dynamic clock.

2. Main menu – a sub-feature under home menu. Displays the current system’s feature and their respective names to be used with the nav command.

3 . Mathematical calculator – A simple tool for performing basic mathematical operations. Now supports logging and lets you access your operation history through integrated database fields.

4. Body Mass Index (BMI) calculator – This feature allows you to estimate your body fat percentage by taking your height and weight value. Height value supports cm, m, ft & in while Weight value supports kg and lbs.

5. Day-Time converter – Instantly convert any time unit (days, hours, minutes, seconds) to another equivalent unit.

6. Temperature converter – Instantly convert temperature values like Celsius (°C) to Fahrenheit (°F) and vice versa.

7. Ping Pong – A simple game of the classic title. You are in for a Player vs COM game. First to score 20 points win the game.

8. ASCII animation test- an experimental feature that showcases a short animation, made in multiple frames of ASCII characters.
